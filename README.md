# Kishore N S — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.  
Live at: **https://Kishorens17.github.io/portfolio**

## Features
- Dark professional theme with animated hero section
- Typed text animation cycling through titles
- Scroll-triggered reveal animations
- Responsive design (mobile, tablet, desktop)
- Sections: About, Education, Skills, Projects, Publications, Awards, Contact
